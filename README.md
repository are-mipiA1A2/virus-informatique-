# virus-informatique- chachou edou a la fin -x belette flo
modélisation de la propagation d'un virus informatique au sein d'un ensemble d'ordinateur et de serveurs reliés en réseau interne et publique.
les paramètres seront entre autres:

- la date de mise en ligne du virus
- le niveau de prtection de chaque usager (anti-virus)
- le niveau de complexité du virus (donc plus ou moin détecté par les anti virus)
- le type de transmission du virus (mail, fichier vérolé, cheval de troie etc)
